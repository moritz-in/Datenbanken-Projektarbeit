`get_products_with_joins(page, page_size)`:
```sql
SELECT p.id AS product_id, p.name, p.price, p.currency,
       b.name AS brand, c.name AS category
FROM products p
LEFT JOIN brands b ON p.brand_id = b.id
LEFT JOIN categories c ON p.category_id = c.id
ORDER BY p.id
LIMIT :limit OFFSET :offset
```
Also: `SELECT COUNT(*) FROM products` for total.
Tags: separate query `SELECT pt.product_id, t.name FROM product_tags pt JOIN tags t ON pt.tag_id = t.id WHERE pt.product_id IN :ids` — build a dict of `{product_id: [tag_names]}` and attach.
Return: `{"items": [...], "total": N}`. Each item: `{product_id, name, brand, category, tags: [], price, currency}`.

`get_dashboard_stats()`:
```sql
SELECT 'products' AS tbl, COUNT(*) AS cnt FROM products
UNION ALL SELECT 'brands', COUNT(*) FROM brands
UNION ALL SELECT 'categories', COUNT(*) FROM categories
UNION ALL SELECT 'tags', COUNT(*) FROM tags
```
Return: `{"mysql_counts": {"products": N, "brands": N, "categories": N, "tags": N}, "last_runs": [...]}`.
Wrap in try/except — on exception return `{"mysql_counts": {"error": str(e)}, "last_runs": []}`.

`get_last_runs(limit=10)`:
```sql
SELECT id, strategy, started_at, finished_at, products_processed, products_written, status
FROM etl_run_log ORDER BY started_at DESC LIMIT :limit
```
Return list of dicts. If table empty, return `[]`. Key `run_timestamp` = `started_at` formatted as string (use `str(row["started_at"])` — SQLAlchemy returns datetime; template uses it as string).

`get_audit_entries(page, page_size)`:
Same as get_last_runs but paginated. Return `{"items": [...], "total": N}`.

`has_column(table, column)`:
```sql
SELECT COUNT(*) AS cnt FROM information_schema.columns
WHERE table_schema = DATABASE() AND table_name = :table AND column_name = :column
```
Return `bool(result[0]["cnt"] > 0)`.

`execute_raw_query(query)`:
Implement `_strip_string_literals(query)` — replace `'...'` content with `''` using `re.sub(r"'[^']*'", "''", query)`.
Implement `_extract_table_names(query)` — extract identifiers after `FROM`/`JOIN` keywords using regex `r'\b(?:FROM|JOIN)\s+(\w+)'` (case-insensitive).
Security: stripped = `_strip_string_literals(query.upper())`; reject if not starting with `SELECT`; reject if any of `INSERT|UPDATE|DELETE|DROP|CREATE|ALTER|TRUNCATE|EXEC` found in stripped.
Execute with `text(query)` (no params — user provided raw SQL). Return `[dict(r) for r in result.mappings()]`.

`load_products_for_index(limit=None, include_tags=True)`:

SELECT p.id, p.name, p.description, p.price, p.currency,

FROM products p
LEFT JOIN brands b ON p.brand_id = b.id
LEFT JOIN categories c ON p.category_id = c.id
[LIMIT :limit if limit provided]
