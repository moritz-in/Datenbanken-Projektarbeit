```python
def call_import_product(self, ...) -> dict:
    with self._get_session() as session:
        conn = session.connection()
        cursor = conn.connection.cursor()  # raw pymysql cursor
        try:
            # Set OUT params as user variables first
            cursor.execute("SET @rc = 0, @rm = ''")
            cursor.execute(
                "CALL import_product(%s, %s, %s, %s, %s, %s, %s, %s, @rc, @rm)",
                (name, description, brand_name, category_name, price, sku, load_class, application)
            )
            # CRITICAL: Flush result sets before reading OUT params
            while cursor.nextset():
                pass
            # Read OUT params
            cursor.execute("SELECT @rc AS result_code, @rm AS result_message")
            row = cursor.fetchone()
            return {"result_code": row[0], "result_message": row[1]}
        finally:
            cursor.close()
```

The procedure must use DELIMITER $$ wrapper (required for multi-statement procedures in MySQL init scripts).

Procedure signature:
```sql
CREATE PROCEDURE import_product(
    IN p_name VARCHAR(500),
    IN p_description TEXT,
    IN p_brand_name VARCHAR(100),
    IN p_category_name VARCHAR(100),
    IN p_price DECIMAL(10,2),
    IN p_sku VARCHAR(100),
    IN p_load_class VARCHAR(20),
    IN p_application VARCHAR(20),
    OUT p_result_code INT,
    OUT p_result_message VARCHAR(500)
)
```

Procedure logic (in order):
1. `DECLARE EXIT HANDLER FOR SQLEXCEPTION BEGIN SET p_result_code = 3; SET p_result_message = 'Datenbankfehler'; END;`
2. Initialize: `SET p_result_code = 0; SET p_result_message = '';`
3. Validation check: IF `TRIM(p_name) = ''` OR `p_price < 0` OR `TRIM(p_category_name) = ''` THEN SET code=2, message='Pflichtfelder fehlen: name, price und category sind erforderlich', LEAVE proc (use a `proc_label: BEGIN...END` label with `LEAVE proc_label`)
4. Duplicate SKU check: `SELECT COUNT(*) INTO @sku_count FROM products WHERE sku = p_sku AND p_sku IS NOT NULL AND p_sku <> ''`; IF @sku_count > 0 THEN SET code=1, message=CONCAT('Doppelte SKU: ', p_sku), LEAVE
5. Resolve brand: `SELECT id INTO @brand_id FROM brands WHERE name = p_brand_name`; IF @brand_id IS NULL use brand_id=1 (fallback to first brand)
6. Resolve category: `SELECT id INTO @cat_id FROM categories WHERE name = p_category_name`; IF @cat_id IS NULL SET code=2, message=CONCAT('Kategorie nicht gefunden: ', p_category_name), LEAVE
7. Insert: `INSERT INTO products (name, description, brand_id, category_id, price, sku, load_class, application) VALUES (...)`
8. SET p_result_code=0, p_result_message=CONCAT('Produkt importiert: ', p_name)

Use `proc_label: BEGIN ... END proc_label` pattern to support LEAVE statements for early exit.

Add DROP PROCEDURE IF EXISTS before CREATE PROCEDURE for idempotency.

```python
@bp.route("/validate/procedure", methods=["GET", "POST"])
def validate_procedure():

    result = None

    if request.method == "POST":
        form_data = {
            "name": request.form.get("name", "").strip(),

            "sku": request.form.get("sku", ""),
            "load_class": request.form.get("load_class", ""),
            "application": request.form.get("application", ""),
        }
        try:
            svc = ServiceFactory.get_product_service()
            result = svc.import_product(
                name=form_data["name"],
                description=form_data["description"],
                brand_name=form_data["brand_name"],
                category_name=form_data["category_name"],
                price=float(form_data["price"] or 0),
                sku=form_data["sku"],
                load_class=form_data["load_class"],
                application=form_data["application"],
            )
        except Exception as e:
            result = {"result_code": 3, "result_message": str(e)}
    return render_template("validate_procedure.html", result=result, form_data=form_data)
```
